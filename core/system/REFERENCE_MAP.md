# 3. RULES

## STRUCTURE RULE
Pages are composed only of predefined blocks.

No new block types may be introduced without updating this file.

---

## CONTENT RULE
Text inside blocks must remain:
- concise
- expert-level
- non-poetic
- non-expansive

---

## DESIGN RULE
Structure is fixed. Content is variable.

Meaning:
- layout does not change
- block types do not change
- only data inside blocks changes
